Fluid Behavior Using Phone and Tablet Classes

Notice that if you omit tablet, the default "fallback" fluid behavior will kick in (your column will essentially become width auto with 20px left and right hand margins). In fact, if you narrow you viewport to about tablet width (~480w-900w), you should see the first four columns exhibiting this below. You'll note that the fallback margins don't quite line upâ€¦

The take-away is if you choose to use phone-X for more control, you will likely want to also use tablet-X. Using phone-X and tablet-X together will likely result in a more predictable transition that you're expecting and left and right hand margins that align.
